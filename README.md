SF作家兼ソフトウェアエンジニア。主な言語は日本語とRuby。  
2018年に [ゲンロン 大森望 SF創作講座](https://school.genron.co.jp/sf/) 3期受講をきっかけに活動開始。  
京都大学大学院情報学研究科修士課程修了。修士（情報学）。
